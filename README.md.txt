# Music Store Sales Analysis (SQL Project)

This SQL project is an in-depth analysis of a digital music store database using PostgreSQL. The goal was to answer real-world business questions related to customers, sales, music genres, and artist performance using SQL queries.

---

# Objective

To perform SQL-based analysis on a music store dataset and derive insights that can help improve customer engagement, optimize marketing strategies, and understand product performance.

---

# Tools Used

- PostgreSQL (via pgAdmin4)
- SQL (Data analysis & reporting)
- GitHub (Version control and portfolio hosting)

---

# Key Business Questions Solved

Here are the main SQL challenges tackled in this project:

1. Who is the senior most employee based on job title?
2. Which countries have the most invoices?
3. What are the top 3 invoice totals?
4. Which city has the best customer (for organizing a promotional music festival)?
5. Who is the best customer (based on total money spent)?
6. Who are the Rock music listeners?
    Return their email, first name, last name, and sort alphabetically by email.
7. Which artists have written the most Rock music?
    Return top 10 rock bands by total track count.
8. What are the songs that are longer than the average length?
    Return the track name and duration (in milliseconds), ordered by length.
9. How much has each customer spent on music by artist?
    Return customer name, artist name, and total amount spent.
10. What is the most popular music genre in each country?
    Based on the number of purchases.
11. Who is the top spending customer in each country?
    Return the country, customer, and total amount spent. If tied, return all.

---

# Files Included

- `music_store_analysis.sql` – Contains all the SQL queries listed above

# Insights Discovered

- Top Rock Artists like Queen and AC/DC dominate the track count and purchases.
- The USA has the highest number of invoices and spending.
- Top customers contribute significantly to revenue — a few customers spend the most.
- Cities with the best customers could be ideal for promotions or events.
- Rock continues to be one of the most popular genres across regions.

---

# How to Use

1. Clone or download this repository.
2. Open `music_store_analysis.sql` in pgAdmin4 or any PostgreSQL interface.
3. Make sure the Music Store database is loaded.
4. Run queries one by one to view the results and insights.
